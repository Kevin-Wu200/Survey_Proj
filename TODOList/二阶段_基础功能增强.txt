====================================================================
二阶段 · 基础功能增强
====================================================================
目标：在主链路闭环基础上，完善各子系统的核心功能，使系统具备
基本的自动化作业能力。

预计工期：第3~5周（与三阶段部分重叠）
依赖：一阶段主链路闭环 ✅

--------------------------------------------------------------------
☑ 2.1 无人机航点任务模块
    - 基于 DJI OSDK API 封装 Waypoint V2 接口
    - 实现：经纬度航点上传 → 航线上传 → 任务启动 → 进度反馈
    - 支持等距拍照触发（相机快门与航点绑定）
    - 仿真中验证多边形/蛇形航线自动执行
    输出：无人机可沿预设航线自主飞行并拍照

☑ 2.2 无人车自主导航
    - 基于 Nav2 集成 SLAM 建图（先用 Cartographer 2D）
    - 配置 Smac Hybrid-A* 全局规划器 + Regulated Pure Pursuit 局部规划器
    - 实现：指定目标点 → 自主路径规划 → 动态避障 → 到达
    - 代价地图参数调优（分辨率 0.05m，宽高 10m×10m）
    输出：无人车可接收导航目标点并自主到达

☑ 2.3 Web 前端功能增强（一）
    - 实现 高德地图 离线瓦片加载（离线地图支持，基于 IndexedDB 缓存）
    - 无人机航线条令下发界面（在高德地图上画航线 → 生成航点 → 发送至 UAV）
    - 无人车导航目标点下发（点击地图 → 发送目标位姿至 UGV）
    - WebSocket 告警推送（电量低 / 连接断开 / 任务完成）
    输出：Web 端可下发航线/航点，接收告警推送

☑ 2.4 任务回放功能
    - 后端：解析 ROS Bag 数据，提取位姿/图像/点云时间序列
    - 前端：自定义时间轴控件，在高德地图上回放 UAV/UGV 历史轨迹
    - 支持倍速回放（1× / 2× / 4× / 8×）
    输出：Web 端可按时间轴回放历史任务

☑ 2.5 仿真/实物模式切换
    - 实现启动时模式选择按钮（仿真 / 实物）
    - 传感器输入接口抽象（仿真传感器 ↔ 实物传感器统一 API）
    - 控制输出接口抽象（仿真执行器 ↔ 实物执行器统一 API）
    输出：同一套算法代码可在仿真/实物模式下切换运行

☑ 2.6 容错机制实现
    - 通信超时检测（3s 超时，最多重试 3 次）
    - 无人车断连 30s → 原地等待，60s → 原路返回
    - 无人机断连 60s → 执行 DJI Return to Home
    输出：手动断开网络可验证各降级策略正确执行
